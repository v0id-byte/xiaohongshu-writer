# 小红书笔记写作公式 Skill（官方企业版）

> 来源：小红书官方企业用户秘籍（2026-04）
> 版本：v1.0

---

## 一、笔记类型（谁可以写笔记）

### 品牌发声
| 类型 | 说明 | 内容方向 |
|------|------|----------|
| 专业号笔记 | 企业的门面，奠定品牌调性 | 品牌故事、行业知识、使用场景、运营活动、用户反馈 |
| 主理人笔记 | 品牌故事分享，增加粉丝粘性 | 创业故事、主理人生活、团队结构、产品设计/工艺 |
| 员工KOS笔记 | 更软性、人感、丰富表述空间 | 工作日常、干货知识、服务案例、品牌幕后 |
| 授权笔记 | 矩阵号提升曝光 | 关联账户内容 |

### 达人发声
| 类型 | 说明 |
|------|------|
| 合作内容笔记（蒲公英） | 实现潜客种草，打开人群认知破圈 |
| 买手笔记 | 快速分销，获取销售规模增长 |

### 素人UGC
- 晒单笔记
- 购后评论

## 二、什么是优质笔记

### 基础要素

| 要素 | 标准 | 创作公式 |
|------|------|----------|
| 标题 | 16-19字 | 「痛点/场景」+「核心买点」= 标题 |
| 正文 | 500-600字（图文）/ >60s（视频） | 热点/场景/痛点 + 简单直接开头 + 引导 |
| 话题 | 4-8个 | 相关性高，贴切主题/产品特点 |
| 组件 | ≤2个 | - |

### 写作标准
- **真实客观**：不虚构、不夸大
- **详细全面**：多角度介绍特点和使用方法
- **个性化**：增加可读性和趣味性
- **内容实用**：「利TA」原则，有参考价值

### 关键指标（自然流量）
- 阅读量 > 1000（近14天）
- 互动数 > 9（点赞+评论+收藏+转发+关注）
- GMV > 200元（近14天）

## 三、写笔记前的思考

写笔记前必须回答4个问题：

1. **这篇笔记能帮谁解决什么问题/提供什么价值？** → 选题
2. **标题和封面是否能让人停下手指点进来？** → 封面&标题
3. **笔记内容是否利于被搜索、被收藏？** → 正文
4. **怎么通过笔记提升用户粘性和信任感？** → 评论区

### 选题原则
- 目标人群广
- 符合商业人群的痛点、兴趣点和热点
- 核心是关键词：长尾词、搜索热词、推荐词

## 四、封面设计（7大元素）

**格式**：3:4（双列笔记下曝光面积最大）

| 元素 | 说明 |
|------|------|
| 关键词 | 突出内容关键词，一目了然，帮助系统识别分类 |
| 成果展示 | 展示使用效果，如"穿在身上效果图"，吸引用户兴趣 |
| 对比 | 前后对比、新旧对比、价格对比，突出差异激发好奇 |
| 情绪 | 加入"开心、惊喜、感动、期待、满足"等情绪，产生共鸣 |
| 原生感 | 生活照、真实场景、日常瞬间、自然光，更容易获得信任 |
| 专业知识 | 解释关键知识点，用疑问句触发用户思考和共鸣 |
| 有用测评 | 突出卖点，创建具体场景/氛围来支持测评 |
| 有趣剧情 | 用真实场景演示产品应用，强调真实同时加入有趣元素 |

## 五、爆款标题结构

| 类型 | 结构 | 示例 |
|------|------|------|
| 数据型 | 具体数据 + 关键词/解决方案 | 100%羊绒拉毛 \| 粉糯羊绒衫，冬日温柔杀 |
| 情绪型 | 情绪词 + 成果 | 煎炸蒸煮样样行，这口不粘锅太万能！ |
| 制造权威型 | 权威信息 + 关键词/解决方案 | 跟着XX学护肤，真的有点东西！ |
| 制造亲密型 | 人物 + 口语化内容 + 感叹号 | 闺蜜！这是适合我们的XXX |
| 说反话/反常识型 | 反转关键词，增加好奇心 | 不是XXX买不起，而是XXX更有性价比 |
| 蹭热点型 | 热门网络话术结构 | XX基础，XX就不基础 |

### 标题要素
- 字数控制 16-19字
- 囊括关键词，突出重点，触达用户真实场景和痛点
- 巧用网络用语、数字、emoji、小红书化语言
- 和用户产生互动感，引发共鸣

## 六、正文写作

### 差正文 vs 好正文
- **差正文**：句式复杂 / 信息量爆炸
- **好正文**：热点/场景/痛点 + 开头简单直接展开 + 引导

### 提升正文感官小妙招
1. **善用标点符号**：`""、/、?、！` 既能分清结构，也能传递感情
2. **善用结构分段**：理清逻辑关系，让用户一眼知道在讲什么

---

## 七、结尾写法

### 目的
- 强化笔记重点
- 引导关注/下单/咨询...

### 要点
1. **简单收个尾**：强化重点
2. **引导评论**：抛出问题或选项
3. **引导动作**：告诉读者为什么要"留住你"

### 示例
> 想要更显瘦的温柔一点的真的可以试试这个v领
> 而且冰岛灰是一个不怎么挑场合和穿搭的颜色，基本秋冬怎么穿都不会出错~
> *(在室内无打光下给你们拍的，所见所得)*

## 八、评论区回复节奏

| 时间 | 行动 |
|------|------|
| 笔记发布后1-2小时 | **高频在线回复**（这段时间评论活跃度对推荐影响较大） |
| 48小时之后 | 不需要每条都回；挑有代表性问题统一回复；对"我也是""好懂"等共鸣型评论适当点赞或简短回复 |
| 高频出现相同评论时 | 可以在后续发一条补充笔记，专门解答集中问题 |

## 九、爆款封面视觉公式

### 对比 + 关键词类
- 尺寸对比（如耳钉6mm/8mm/10mm/12mm/14mm）
- 高低价对比（几百 VS 几万）
- 效果前后对比

### 有趣剧情类
- 创意产品呈现（如首饰放在奶茶杯里）
- 生活方式美学（零食柜、收纳等）

## 十、商品笔记特殊规则

### 流量特点
- 曝光少但精准
- 同时考核互动价值 + 交易价值

### 商品组件显示条件
- **只有「千帆-商品推广」广告流量下才显示商品组件**
- 其他营销目标及自然流量下**无商品组件**

### 商品笔记展示要素
- 标题/正文要有具体场景和痛点
- 至少3张产品图（不同角度）
- 挂商品链接
- 可配合电商资源和活动

## 使用指南

当需要写小红书笔记时，按照以下步骤：

1. **确定笔记类型**（品牌/达人/KOS/UGC）
2. **选题**（目标人群广 + 痛点/热点）
3. **设计封面**（7大元素，3:4格式）
4. **写标题**（6种爆款结构选一个）
5. **写正文**（简洁分段，善用标点）
6. **写结尾**（强化重点 + 引导动作）
7. **评论区运营**（1-2小时高频回复）
