# 小红书笔记写作公式

> 基于小红书官方企业用户秘籍整理的笔记写作 Skill for OpenClaw/Melt 🫠

[![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)

**xiaohongshu-writer** 是一个基于小红书官方企业用户秘籍整理的 AI 写作 Skill，帮助品牌运营者、博主和 AI Agent 快速产出符合平台规范的优质笔记。

涵盖完整的笔记创作公式：选题法 → 封面设计 → 标题结构 → 正文写法 → 结尾引导 → 评论运营 → 商品笔记特殊规则。

## 🚀 快速开始

### 1. 安装 Skill（OpenClaw 平台）

将 Skill 添加到你的 OpenClaw 工作区：

```bash
# 方法一：克隆到本地 skills 目录
git clone https://github.com/v0id-byte/xiaohongshu-writer.git ~/你的skills路径/xiaohongshu-writer

# 方法二：在 OpenClaw 中直接引用
# 当需要写小红书笔记时，直接对 Melt 说：
# "帮我写一篇小红书笔记，关于[产品/主题]，目标是[目标人群]"
```

### 2. 开始写笔记

当需要写笔记时，直接告诉 Melt：

> "帮我写一篇小红书笔记，关于[产品/品牌/主题]"

Melt 会按照官方秘籍公式自动完成：
1. 确定笔记类型（品牌/达人/KOS/UGC）
2. 选题（4问选题法）
3. 设计封面（7大元素 + 3:4格式）
4. 写标题（6种爆款结构）
5. 写正文（500-600字）
6. 写结尾（引导公式）
7. 评论运营节奏

## 📚 详细介绍

### 笔记类型体系

| 类型 | 说明 | 内容方向 |
|------|------|----------|
| **品牌发声** | 专业号笔记、主理人笔记、员工KOS笔记、授权笔记 | 品牌故事、行业知识、创业故事、工作日常 |
| **达人发声** | 合作内容笔记（蒲公英）、买手笔记 | 潜客种草、人群破圈、销售增长 |
| **素人UGC** | 晒单笔记、购后评论 | 真实体验、用户反馈 |

### 优质笔记标准

| 要素 | 标准 |
|------|------|
| 标题 | 16-19字 |
| 正文 | 500-600字（图文）/ >60s（视频）|
| 话题 | 4-8个 |
| 组件 | ≤2个 |

### 写笔记前4问选题法

1. **这篇笔记能帮谁解决什么问题/提供什么价值？** → 选题
2. **标题和封面是否能让人停下手指点进来？** → 封面&标题
3. **笔记内容是否利于被搜索、被收藏？** → 正文
4. **怎么通过笔记提升用户粘性和信任感？** → 评论区

### 封面7大元素（3:4格式）

| 元素 | 说明 |
|------|------|
| 关键词 | 突出内容关键词，一目了然，帮助系统识别分类 |
| 成果展示 | 展示使用效果，如"穿在身上效果图"，吸引用户兴趣 |
| 对比 | 前后对比、新旧对比、价格对比，突出差异激发好奇 |
| 情绪 | 加入"开心、惊喜、感动、期待、满足"等情绪，产生共鸣 |
| 原生感 | 生活照、真实场景、日常瞬间、自然光，更容易获得信任 |
| 专业知识 | 解释关键知识点，用疑问句触发用户思考和共鸣 |
| 有趣剧情 | 用真实场景演示产品应用，强调真实同时加入有趣元素 |

### 6种爆款标题结构

| 类型 | 结构 | 示例 |
|------|------|------|
| 数据型 | 具体数据 + 关键词/解决方案 | 100%羊绒拉毛 \| 粉糯羊绒衫，冬日温柔杀 |
| 情绪型 | 情绪词 + 成果 | 煎炸蒸煮样样行，这口不粘锅太万能！ |
| 制造权威型 | 权威信息 + 关键词/解决方案 | 跟着XX学护肤，真的有点东西！ |
| 制造亲密型 | 人物 + 口语化内容 + 感叹号 | 闺蜜！这是适合我们的XXX |
| 说反话/反常识型 | 反转关键词，增加好奇心 | 不是XXX买不起，而是XXX更有性价比 |
| 蹭热点型 | 热门网络话术结构 | XX基础，XX就不基础 |

### 正文写法公式

- **差正文**：句式复杂 / 信息量爆炸
- **好正文**：热点/场景/痛点 + 开头简单直接展开 + 引导

**小妙招**：
1. 善用标点符号：`""、/、?、！` 既能分清结构，也能传递感情
2. 善用结构分段：理清逻辑关系，让用户一眼知道在讲什么

### 结尾引导公式

1. **简单收个尾**：强化笔记重点
2. **引导评论**：抛出问题或选项
3. **引导动作**：告诉读者为什么要"留住你"

### 评论区回复节奏

| 时间 | 行动 |
|------|------|
| 笔记发布后1-2小时 | **高频在线回复**（这段时间评论活跃度对推荐影响较大） |
| 48小时之后 | 不需要每条都回；挑有代表性问题统一回复；对共鸣型评论适当点赞或简短回复 |
| 高频出现相同评论时 | 可以在后续发一条补充笔记，专门解答集中问题 |

### 商品笔记特殊规则

- 曝光少但精准，同时考核**互动价值 + 交易价值**
- 只有「千帆-商品推广」广告流量下才显示商品组件
- 其他营销目标及自然流量下**无商品组件**
- 商品笔记要素：标题/正文要有具体场景和痛点、至少3张产品图、挂商品链接、可配合电商资源和活动

## 💡 使用示例

### Prompt 示例

```
请为 [某品牌羊绒衫] 写一篇小红书笔记：

- 产品：100%羊绒拉毛，粉糯质感
- 目标人群：25-35岁都市女性
- 核心卖点：冬日温柔、显瘦、适合多种场合
- 痛点：冬天穿搭臃肿、不知道怎么选羊绒

要求：
1. 用情绪型标题结构
2. 封面加入对比+情绪元素
3. 正文500-600字，包含场景化开头和产品卖点
4. 设计2个能引发讨论的评论引导问题
5. 搭配4-8个相关话题
```

### 输出示例

```
## 标题
煎炸蒸煮样样行，这件羊绒衫太万能！

## 正文
（按照公式自动生成500-600字正文，包含热点场景开头、产品卖点、真实体验、结尾引导）

## 话题
#羊绒衫 #冬季穿搭 #显瘦穿搭 #都市女性 ...

## 评论引导问题
- "你们喜欢温柔色还是高级灰？"
- "你觉得羊绒衫搭配裤装好看还是裙装好看？"
```
